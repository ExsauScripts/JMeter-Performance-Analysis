# JMeter Performance Analysis 📊

Load testing with Apache JMeter combined with a Python script that 
automatically parses the results and generates a full performance report.

---

## How it works

1. JMeter runs the load test and exports results as a CSV
2. The Python script reads that CSV and generates:
   - A formatted **Excel report** with summary + full detail tabs
   - A **bar chart** showing response time per request with average line

---

## Files

| `Scripts/Test_Plan(karaokeivan).jmx` | JMeter test plan (load test on karaokeivan.com) |
| `Scripts/Analisis_jmeter.py` | Python script that parses results and generates report |
| `Results/Final Results.csv` | Sample JMeter output used as input for the analysis |
| `Results/GraficoRendimiento.png` | Sample output chart generated by the script |

---

## How to run

### 1. Install dependencies
```bash
pip install pandas matplotlib xlsxwriter
```

### 2. Run the analysis
```bash
python Scripts/Analisis_jmeter.py
```

## Tech Stack

| Apache JMeter | Load test execution
| Python | Results parsing and report generation
| pandas | CSV processing and Excel export
| matplotlib | Chart generation
| xlsxwriter | Excel formatting
